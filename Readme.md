Install
---

    $ component install kelonye/ember-ckedit

Example
---

    $ make


Api
---

Mixin

    Return mixin

Mixin#for

    the view context's attr which the editor's data will be set on creation.

Mixin#isInline

    set to true to make editor inline

Mixin#editorOpts

    editor options

Mixin#events

    array of events to trigger value update
